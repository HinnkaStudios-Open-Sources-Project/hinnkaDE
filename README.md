# Start The Installation:

The [bash](https://github.com/kk2lly/hinnkaDE/blob/main/hinnkaDE.sh) script must be to `/usr/local/bin` or `/bin`

The [wallpaper image](https://github.com/kk2lly/hinnkaDE/blob/main/Wallpaper.png) also must be to `/usr/local/bin` or `/bin`

The [desktop session](https://github.com/kk2lly/hinnkaDE/blob/main/hinnkaDE.desktop) file must be to `/usr/share/xsession`

Once done, log out to change desktop session to ''hinnkaDE'', log on and done!
